## Installation

1. Clone the repository:
   git clone https://github.com/cristiancavelasquez/Recipe-page-Astro.git

2. Navigate to the project directory:
   cd Recipe-page-Astro

3. Install dependencies:
   npm install

4. Start the development server:
   npm run dev
